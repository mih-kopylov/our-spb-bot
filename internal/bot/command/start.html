Привет! Этот бот делает то же, что официальное приложение портала https://gorod.gov.spb.ru, но с дополнительными опциями. Упрощённые категории Официальное приложение содержит огромное количество категорий, поиск в котором затруднителен. Чаще всего мы используем около 10-15 наиболее распространённых категорий. Бот позволяет иметь свой список категорий. Текст сообщения по умолчанию В большинстве случаев текст обращения одинаковый. Бот позволяет задать текст по умолчанию для каждой категории. При отправке сообщения его можно поменять, если потребуется. Очередь сообщений Все сообщения, отправленные боту, добавляются в очередь и отправляются при первой возможности. В случаях, когда сервер приложения сбоит, на обслуживании или просто не работает, а также в случаях, когда превышен суточный лимит обращений на пользователя, бот отправит обращения, как только сможет, самостоятельно. Несколько аккаунтов На портале действует ограничение по количеству обращений в сутки. Если в вашей семье у каждого есть аккаунт на портале, вы можете добавить бота в общую группу и добавить ему несколько аккаунтов портала. В этом случае бот будет брать обращения из общей очереди и отправлять их всеми доступными аккаунтами, таким образом, увеличивая суточный лимит. Геолокация Бот берёт текущую геолокацию устройства для обращения, таким образом, нет необходимости её выбирать вручную. Для некоторых категорий, где требуется адрес дома, выбирается дом, ближайший к текущему местоположению.
